String Escape Sequences:
\n - New Line
>>> new_line = "hello\nthere"
>>> print(new_line)
hi
there

\\ - Prints a backslash
>>> str = "this is a backslash: \\"
>>> print(str)
this is a backslash: \

\" - Prints a double quote
\' - Prints a single quote




 # Set the message variable equal to any string containing a new-line escape sequence
